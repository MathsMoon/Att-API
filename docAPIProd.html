Documentação da API REST

Seção: Carros

1. Listar Todos os Carros

Método: GET

Rota: /carros

Retorna uma lista de todos os carros cadastrados no banco de dados.

Exemplo de resposta:
[
  {
    "id": 1,
    "nome": "Fusca",
    "modelo": "Classic",
    "ano": 1974,
    "preco": 15000.0
  },
  {
    "id": 2,
    "nome": "Gol",
    "modelo": "G5",
    "ano": 2010,
    "preco": 20000.0
  }
]

2. Listar um Carro Específico

Método: GET

Rota: /carros/<int:carro_id>

Retorna os dados de um carro específico pelo seu ID.

Exemplo de resposta:
{
  "id": 1,
  "nome": "Fusca",
  "modelo": "Classic",
  "ano": 1974,
  "preco": 15000.0
}

3. Criar um Novo Carro

Método: POST

Rota: /carros

Cria um novo carro com base nos dados fornecidos no corpo da requisição.

Exemplo de corpo da requisição (JSON):
{
  "nome": "Fusca",
  "modelo": "Classic",
  "ano": 1974,
  "preco": 15000.0
}

Exemplo de resposta:
{
  "id": 3,
  "nome": "Fusca",
  "modelo": "Classic",
  "ano": 1974,
  "preco": 15000.0
}

4. Atualizar um Carro

Método: PUT

Rota: /carros/<int:carro_id>

Atualiza os dados de um carro específico pelo seu ID.

Exemplo de corpo da requisição (JSON):
{
  "nome": "Fusca Atualizado",
  "modelo": "Classic",
  "ano": 1974,
  "preco": 16000.0
}

Exemplo de resposta:
{
  "id": 1,
  "nome": "Fusca Atualizado",
  "modelo": "Classic",
  "ano": 1974,
  "preco": 16000.0
}

5. Deletar um Carro

Método: DELETE

Rota: /carros/<int:carro_id>

Remove um carro específico do banco de dados pelo seu ID.

Exemplo de resposta:
{
  "mensagem": "Carro excluído com sucesso"
}

Seção: Usuários

6. Listar Todos os Usuários

Método: GET

Rota: /usuarios

Retorna uma lista de todos os usuários cadastrados no banco de dados.

Exemplo de resposta:
[
  {
    "id": 1,
    "nome": "João Silva",
    "email": "joao@exemplo.com"
  },
  {
    "id": 2,
    "nome": "Maria Oliveira",
    "email": "maria@exemplo.com"
  }
]

7. Listar um Usuário Específico

Método: GET

Rota: /usuarios/<int:usuario_id>

Retorna os dados de um usuário específico pelo seu ID.

Exemplo de resposta:
{
  "id": 1,
  "nome": "João Silva",
  "email": "joao@exemplo.com"
}

8. Criar um Novo Usuário

Método: POST

Rota: /usuarios

Cria um novo usuário com base nos dados fornecidos no corpo da requisição.

Exemplo de corpo da requisição (JSON):
{
  "nome": "Pedro Oliveira",
  "email": "pedro@exemplo.com",
  "senha": "senhaSegura123"
}

Exemplo de resposta:
{
  "id": 3,
  "nome": "Pedro Oliveira",
  "email": "pedro@exemplo.com"
}

9. Atualizar um Usuário

Método: PUT

Rota: /usuarios/<int:usuario_id>

Atualiza os dados de um usuário específico pelo seu ID.

Exemplo de corpo da requisição (JSON):
{
  "nome": "Pedro Oliveira Atualizado",
  "email": "pedro@atualizado.com",
  "senha": "novaSenha123"
}

Exemplo de resposta:
{
  "id": 3,
  "nome": "Pedro Oliveira Atualizado",
  "email": "pedro@atualizado.com"
}

10. Deletar um Usuário

Método: DELETE

Rota: /usuarios/<int:usuario_id>

Remove um usuário específico do banco de dados pelo seu ID.

Exemplo de resposta:
{
  "mensagem": "Usuário excluído com sucesso"
}
